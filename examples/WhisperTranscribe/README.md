# WhisperTranscribe

Speech-to-text with a pretrained [OpenAI Whisper](https://huggingface.co/openai/whisper-tiny)
checkpoint -- the first **audio** example in the library. The pipeline is
pure CAI Pascal end to end:

1. **WAV reader + log-mel frontend** (`neural/neuralaudio.pas`): 16 kHz mono
   16-bit PCM in, the exact HF `WhisperFeatureExtractor` log-mel spectrogram
   out (400-point periodic-hann STFT, hop 160, 80 slaney-scale mel bins,
   `log10`, global max-8 clamp, `(x+4)/4`; verified against the HF oracle to
   ~1e-5 by `TestWhisperLogMelFrontend`).
2. **Whisper encoder** (`BuildWhisperFromSafeTensors` in
   `neural/neuralpretrained.pas`): Conv1d+GELU x2 (the stride-2 conv halves
   3000 mel frames to 1500 hidden states), fixed sinusoidal positions,
   pre-norm transformer blocks, final LayerNorm.
3. **Whisper decoder**: learned positions, causal self-attention +
   cross-attention over the 1500-frame encoder states
   (`TNNetCrossAttention`, rectangular scores), tied LM head -- greedily
   autoregressed from the prologue
   `<|startoftranscript|><|en|><|transcribe|><|notimestamps|>`.
4. **Byte-level BPE detokenization** (`neural/neuralhftokenizer.pas`).

## Usage

Download the checkpoint (about 151 MB):

```bash
mkdir -p /tmp/whisper-tiny && cd /tmp/whisper-tiny
wget https://huggingface.co/openai/whisper-tiny/resolve/main/model.safetensors
wget https://huggingface.co/openai/whisper-tiny/resolve/main/config.json
wget https://huggingface.co/openai/whisper-tiny/resolve/main/tokenizer.json
```

Build and run (any 16 kHz mono 16-bit PCM WAV up to 30 s; convert with
`ffmpeg -i in.ext -ar 16000 -ac 1 out.wav`):

```bash
fpc -Fu../../neural -Mobjfpc -Sh -O2 WhisperTranscribe.lpr
./WhisperTranscribe /tmp/whisper-tiny speech.wav
```

Verified output on the classic `jfk.wav` sample (whisper.cpp's test file)
with `openai/whisper-tiny`:

```
Transcription: And so my fellow Americans ask not what your country can do
for you ask what you can do for your country.
```

Resources: about 2.5 GB resident memory at whisper-tiny scale (if you cap
the address space, allow `ulimit -v 4000000` -- the 3 GB cap used for the
test binaries is too tight for this demo) and roughly 5 minutes end to end
on CPU (build ~2 min, encode ~1 min, 23 greedy decode steps ~2 min; each
step re-runs the full decoder -- no KV cache on this two-net path).

The English-only `*.en` checkpoints work too: they have no language/task
special tokens and the prologue degrades gracefully to the subset present
in the vocabulary.

The pico parity fixture for the importer lives in
`tests/fixtures/tiny_whisper.*` (generated by
`tools/whisper_tiny_fixture.py`); `TestWhisperParity` asserts encoder
hidden states and decoder logits against the float64 HF oracle.
